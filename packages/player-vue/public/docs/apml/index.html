APML Specifications

Accelerated Production Markup Language

APML is the specification language for the SSi learning system. It defines what we build, why we build it, and how the pieces fit together.

These specs are the source of truth. Code implements the spec; the spec doesn't document the code.

ğŸ“– APML Design Principles

Code Principle
D01 Spec first, code second. The spec is the source of truth.
D02 Language-agnostic. Works for Welsh, Japanese, Arabic, anything.
D03 Everything is a parameter. No magic numbers in code.
D04 Cognitive load over word count. Syllables, not words.
D05 Store raw, compute derived. Don't persist calculations.
D06 The LEGO is the unit. Spaced rep tracks LEGOs, not phrases.
D07 Context-agnostic reuse. Same audio in different contexts.
