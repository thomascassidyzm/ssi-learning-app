Platform: Chrome PWA

Choice Rationale
Chrome-only PWA Schools use Chromebooks; single browser target simplifies
Supabase Realtime Live updates for "who's learning now" dashboard
Dark/Light Theme Teacher preference; classroom lighting varies
SSi Brand Aesthetic Consistent with learner app; professional appearance

Design System

  • Colors: SSi Red (#c23a3a), SSi Gold (#d4a853), Dark backgrounds
  • Fonts: Noto Sans JP (display), Source Sans 3 (body)
  • Components: Cards, tables, charts, belt icons, sparklines
  • Animations: Subtle fade-in, hover states, progress bars
